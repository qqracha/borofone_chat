# Обзор

### О чем проект?

### Обзор

Этот проект —  “Discord-подобный” чат: комнаты, сообщения, HTTP-эндпоинты и WebSocket-стрим новых сообщений.

### Что есть сейчас

* Healthcheck, который проверяет доступность Postgres и Redis.&#x20;
* Создание комнаты и получение списка сообщений комнаты по HTTP.
* WebSocket-комната с рассылкой события `message.new`&#x20;
